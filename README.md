# InVision

InVision was a digital product design platform used by teams to build customer
experiences through rapid prototyping, design collaboration, and design system
management. It provided REST APIs for managing prototypes, design documents,
boards, comments, workflows, and design system components.

**Note: InVision permanently shut down all services on January 1, 2025.**
The invisionapp.com domain now redirects to Miro.

## APIs

- **DSM Design Tokens API** — Retrieve design tokens (colors, text styles,
  fonts) and icons from Design System Manager (DSM) projects via API key
  authentication at `https://projects.invisionapp.com/dsm-export/`
